개인정보처리방침

최종 업데이트: 2024년 12월 7일

본 개인정보처리방침은 Caesar Apps(이하 "회사")가 제공하는 모바일 애플리케이션(이하 "앱")의 개인정보 처리에 관한 사항을 안내합니다.

1. 수집하는 정보

본 앱은 사용자의 개인정보를 직접 수집하지 않습니다. 다만, 다음과 같은 정보가 자동으로 수집될 수 있습니다:

  • 기기 정보: 기기 모델, 운영체제 버전, 고유 기기 식별자
  • 앱 사용 데이터: 앱 사용 빈도, 기능 사용 패턴
  • 광고 식별자: Google 광고 ID (광고 표시 목적)

2. 정보 사용 목적

수집된 정보는 다음 목적으로 사용됩니다:

  • 앱 기능 제공 및 개선
  • 맞춤형 광고 표시 (Google AdMob)
  • 앱 사용 통계 분석
  • 오류 진단 및 수정

3. 제3자 서비스

본 앱은 다음 제3자 서비스를 사용합니다:

  • Google AdMob: 광고 표시를 위해 사용됩니다. Google 개인정보처리방침
  • Google Analytics: 앱 사용 분석을 위해 사용될 수 있습니다.

4. 데이터 저장

앱 내 데이터(설정, 기록 등)는 사용자의 기기에 로컬로 저장됩니다. 회사는 사용자의 개인 데이터를 외부 서버에 저장하지 않습니다.

5. 아동의 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 개인정보를 제공한 사실을 알게 된 경우, 해당 정보를 즉시 삭제합니다.

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 앱 삭제를 통한 로컬 데이터 삭제
  • 기기 설정에서 광고 ID 재설정 또는 맞춤 광고 비활성화
  • 개인정보 관련 문의 및 요청

7. 개인정보처리방침 변경

본 개인정보처리방침은 법률 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경 사항이 있을 경우 앱 내 공지 또는 업데이트를 통해 안내합니다.
